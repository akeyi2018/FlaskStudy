# FlaskStudy
Flask学習

#環境設定(Python3+Ubuntu)(インストール）  
```
$ mkdir projectname  
$ cd projectname  
$ python3 -m vent vent  
$ . venv/bin/activate
(venv) $ pip install Flask
(venv) $ export FLASK_APP=target.py
(venv) $ flask run
```
