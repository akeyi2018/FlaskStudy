# FlaskStudy
Flask学習

##### 環境設定(Python3+Ubuntu)(インストール）  
```
$ mkdir projectname  
$ cd projectname  
$ python3 -m vent vent  
$ . venv/bin/activate
(venv) $ pip install Flask
```  
##### 開発手順  
```
$ cd projectname   
$ . venv/bin/activate
(venv) $ export FLASK_APP=hello.py
(venv) $ flask run
```
