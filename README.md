# FlaskStudy
Flask学習

##### 環境設定(Python3+Ubuntu)(インストール）  
```Text
$ mkdir projectname  
$ cd projectname  
$ python3 -m venv venv  
$ . venv/bin/activate
(venv) $ pip install Flask
```  
##### 開発手順  
```YAML
$ cd projectname   
$ . venv/bin/activate
(venv) $ git clone http://github.com/akeyi2018/FlaskStudy
(venv) $ cd FlaskStudy
```  
##### グラフ表示(Chat.js使用)を追加  
```Text
(venv) $ python Showchart.py
```

##### GET/POSTテスト    
```Text
(venv) $ python hello.py
```

##### 天気予報(3日間)スクレイピングを追加  
```Text
(venv) $ python weather.py
```

##### 天気予報(週間)（地域選択機能）を追加  
```Text
(venv) $ python ShowWeekWeather.py
```
